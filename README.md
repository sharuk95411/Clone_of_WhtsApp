# 💬 WhatsApp Clone (Android)

An Android messaging app clone featuring multiple authentication methods, built to explore Firebase-based real-time login flows.

## ✨ Features
- Login via Gmail, Facebook, or Phone Number
- Firebase Authentication integration
- Native Android UI (XML layouts)

## 🛠️ Tech Stack
`Java` · `Android XML` · `Firebase Authentication`

## 🚀 Getting Started
1. Clone the repo and open in Android Studio
2. Connect your own Firebase project and add `google-services.json`
3. Build and run on an emulator or device
